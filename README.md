# ITI JS Project

Simple JavaScript project scaffold created from the existing workspace.

Quick start

1. Install a static server if needed, for example `npm install -g http-server` or use `npx`.
2. Serve the project:

```
npx http-server -c-1 . -p 8080
```

Open `http://localhost:8080/src/html/index.html` in your browser.

To push to GitHub:

```
cd "ITI JS Project"
git remote add origin <your-remote-url>
git push -u origin main
```
